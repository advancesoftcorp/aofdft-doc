===================================================
深層学習DFTソフトウェア Advance/OF-DFT
===================================================

Advance/OF-DFTは、深層学習運動エネルギー汎関数AdvanceSoft26を搭載し、Orbital-Free DFT (OF-DFT)の拡張理論であるAtomic Orbital Facilitated DFT (AOF-DFT)を実装した電子状態計算ソフトウェアです。 

AdvanceSoft26を用いたAOF-DFT計算では、従来のOF-DFTよりも高精度かつ汎用的に、Kohn-Sham DFTよりも高速にシミュレーションを行うことが可能です。
